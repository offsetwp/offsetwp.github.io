## Installation

```bash
git clone git@github.com:offsetwp/offsetwp.github.io.git
cd offsetwp.github.io
nvm use # 24.8.0
npm install
```

## Run development

```bash
nvm use
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
